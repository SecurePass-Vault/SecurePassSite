## HOW IT WORKS SECTION
**4 Steps in timeline:**

1. **Setup Your Vault**
   Set master password. System generates recovery key and creates encrypted vault file with dual key protection.

2. **Store Your Passwords**
   Add, organize, and manage passwords with associated details like website URLs, usernames, and secure notes.

3. **Automatic Protection**
   Application auto-locks after inactivity. All data remains encrypted in vault file at all times.

4. **Recovery Options**
   Forgot master password? Use recovery key to regain access or set new password without data loss.
